# Programming Visualization

一个用于整理“编程可视化 / 代码执行可视化 / 算法动画 / Agent 执行轨迹可视化”相关实验的项目目录。

## 初始方向

- 代码执行过程可视化
- 算法与数据结构动画
- LLM / Agent 工具调用轨迹可视化
- Shell、Git、Python 脚本执行流程展示
- 面向教学视频或交互式 Demo 的可视化组件

## 目录规划

```text
programming-visualization/
├── README.md
├── docs/
├── examples/
├── src/
└── assets/
```
